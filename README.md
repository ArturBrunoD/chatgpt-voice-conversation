# 🎙️ Conversa por Voz com ChatGPT + Whisper + gTTS

## 📌 Descrição
Este projeto implementa um assistente de voz que utiliza:
- **Whisper** (OpenAI) – para converter fala em texto (Speech-to-Text)
- **ChatGPT** (OpenAI) – para gerar respostas inteligentes
- **gTTS** (Google Text-to-Speech) – para transformar a resposta em áudio

O usuário fala uma pergunta, o sistema transcreve, envia ao ChatGPT e devolve a resposta falada.

### Pré‑requisitos
- Python 3.8+

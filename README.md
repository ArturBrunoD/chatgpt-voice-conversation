# 🎙️ Conversa por Voz com ChatGPT + Whisper + gTTS

## 📌 Descrição
Assistente de voz que:
- Grava sua pergunta
- Transcreve com **Whisper** (OpenAI)
- Responde com **ChatGPT**
- Lê a resposta em português com **gTTS**
